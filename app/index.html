Pearson Quarks

Pearson brand assets for software development

During UI development in Pearson English we are following Atomic Design philosophy. Dealing with large scale projects we discovered that we need low level UI library that will provide brand convergence between web, desktop and native mobile apps. Quarks are brand assets that can be used across platforms and technologies and a single point of reference.

Quarks are available on NPM as pearson-quarks.

Quick project bootsraping

Get all brand resources you need on start of digital product development. Save your time on visual assets format conversion and searching for information.

Works for web & native

Use with web and native apps. Assets and the code is available in formats and languages needed for all platforms. Quarks donâ€™t have any dependency to other libraries and frameworks.

Simple & easy to use

Project is public as pearson brand is. Share it with developers and designers by passing a link. Install library via package manager or just download it from github.

Horizontal Logo

The horizontal logo is the preferred choice for digital uses to ensure maximum standout in small areas. Our preferred position of the logo is in the top left-hand side, but when content, context, and usability of your digital touchpoint restricts this (e.g., mobile), use your best judgment to achieve logo standout.
